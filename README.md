A premium music experience built with React, Tailwind CSS, and Node.js.

## Features
- **Cinematic Moods**: Explore music by mood (Love, Soul, Gym, etc.).
- **Master Chart**: View the top tracks in a high-fidelity leaderboard layout.
- **Atmospheric UI**: RetroPulse aesthetic with smooth animations and cinematic imagery.
- **Smart Filter**: Automatically filters out specific content (personalization).
- **Pro Player**: Full-featured player with skip, volume control, and progress tracking.

## Tech Stack
- **Frontend**: React, Vite, Framer Motion, Lucide React, Zustand.
- **Backend**: Node.js, Express, MongoDB, Axios (JioSaavn API integration).
- **Styling**: Tailwind CSS (PostCSS).

## Setup
1. **Backend**:
   ```bash
   cd backend
   npm install
   npm start
   ```
2. **Frontend**:
   ```bash
   cd frontend
   npm install
   npm run dev
   ```

## Folder Structure
- `/frontend`: React application.
- `/backend`: Node.js server.
- `/backend/api`: External music engine integrations.
